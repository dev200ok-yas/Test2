Commandes Git

'git init' => Initialise un nouveau dépôt Git
'git add ' => Ajoute un fichier spécifique à l'index
'git add .' => Ajoute tous les fichiers du répertoire courant à l'index
'git status' => Affiche l'état actuel du dépôt (fichiers modifiés, ajoutés, etc.)
'git diff' => Affiche les différences entre les fichiers modifiés et l'index
'git commit -m "message"' => Crée un commit avec un message
'git commit' => Crée un commit (ouvre un éditeur pour le message)
'git push -u origin ' => Pousse les commits locaux vers un dépôt distant et définit la branche de suivi
'git pull origin ' => Récupère et fusionne les modifications depuis le dépôt distant
'git branch' => Liste les branches locales
'git branch ' => Crée une nouvelle branche
'git checkout ' => Change de branche
'git checkout -b ' => Crée une branche et change dessus
'git log' => Affiche l'historique des commits
'git log --oneline' => Affiche l'historique des commits en une seule ligne par commit
'git remote -v' => Affiche les dépôts distants configurés
'git clone ' => Clone un dépôt distant sur votre machine
'git stash' => Sauvegarde les modifications non commités temporairement
'git stash apply' => Restaure les modifications sauvegardées par git stash
'git rm ' => Supprime un fichier du répertoire de travail et de l'index
